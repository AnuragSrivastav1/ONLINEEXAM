      Online Examination System is a MCQ(Multiple Choice  
   Question) based examination system which is built using    
   JAVA and MySQL. It provides an easy to use Interface for  
    both Test Conductor and students appearing for the  
    examination.   
     
 
Mission
  
     
 
   Responses by the candidates will be checked  
     automatically and instantly. 
   Online examination will reduce the hectic job of  
     assessing the answers given by the candidates. 
   This project would be very useful for educational 
     institutes where regular evaluation of students is  
      required.
